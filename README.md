# Eryawan Portfolio

Portfolio pribadi berbasis **Next.js + Tailwind CSS**.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Lucide React icons

## Menjalankan Lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Build Production

```bash
npm run build
npm start
```

## Customisasi

- Konten utama: `app/page.tsx`
- Styling global: `app/globals.css`
- Tema Tailwind: `tailwind.config.ts`
